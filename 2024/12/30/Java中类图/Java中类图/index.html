PlantUML 代码

在线uml链接：https://uml.zhiyuanye.com/uml/SyfFKj2rKt3CoKnELR1Io4ZDoSa70000

系统描述

  1. 车辆（Vehicle）

    • 基类，包含车辆的通用属性和方法。

  2. 汽车（Car）

    • 继承自车辆（继承）。

  3. 电动车（ElectricCar）

    • 继承自车辆，并实现充电接口（Chargeable）（实现）。

  4. 引擎（Engine）

    • 与汽车相关联（关联）。

  5. 车库（Garage）

    • 包含车辆的集合（聚合）。

  6. 车主（Owner）

    • 与车库绑定的组成部分（组合）。

图解

  1. 继承（Inheritance）

    • Car 和 ElectricCar 继承了 Vehicle，继承关系用实线三角箭头表示。

  2. 实现（Implements）

    • ElectricCar 实现了 Chargeable 接口，用虚线三角箭头表示。

  3. 关联（Association）

    • Car 和 Engine 是关联关系，表示汽车需要一个引擎。

  4. 聚合（Aggregation）

    • Garage 和 Vehicle 是聚合关系，表示车库包含车辆，但车辆可以独立于车库存在。

  5. 组合（Composition）

    • Owner 和 Garage 是组合关系，表示车主的车库绑定于车主的生命周期。
