  • Triplane：1. Tri-Plane 方法的核心思想

传统的 3D 生成方法（如 NeRF）通常使用 体素（Voxel） 或 点云（Point Cloud） 来表示 3D 场景，但这些方法计算量大，渲染速度慢。Tri-Plane 方法的主要特点是：
降低计算复杂度：用 3 个 2D 平面（XY、XZ、YZ） 表示 3D 场景，比直接用 3D 体素要高效得多。
高效查询 3D 信息：任何 3D 空间点的特征，可以通过从这 3 个 2D 平面中双线性插值（bilinear interpolation）得到，从而快速获取 3D 场景信息。
Tri-Plane 具体实现
三平面表示（Triplane Representation）
3D 空间被投影到 XY、XZ 和 YZ 三个正交的 2D 平面，每个平面存储 3D 场景的特征图。对于 3D 空间中的任意一点 (x, y, z)，可以：

  • 从 XY 平面上查找 (x, y) 位置的特征
  • 从 XZ 平面上查找 (x, z) 位置的特征
  • 从 YZ 平面上查找 (y, z) 位置的特征
    将这三个特征进行加权求和或拼接，得到该点的最终特征
    隐式 3D 解码（Implicit Decoder）
    得到点的 3D 特征后，通过一个MLP（多层感知机） 进行解码，输出该点的颜色、密度等信息，完成 3D 场景重建。
    训练与优化
    Tri-Plane 可以与 NeRF、3D GAN、扩散模型等方法结合，通过对比真实图像进行优化，使生成的 3D 结果更加逼真。
  • SSC
    Semantic Scene Completion（SSC，语义场景补全）
    语义场景补全（Semantic Scene Completion，简称 SSC）是一项计算机视觉和 3D 视觉任务，旨在从部分观测数据中推测完整的 3D 场景，并同时进行语义分割。这在自动驾驶、机器人导航、增强现实（AR）和 3D 视觉感知等领域非常重要。

SemCity: 模型方法

1. 概述

SemCity 是一种基于 Triplane 表示 + 扩散模型（Diffusion Model） 的 3D 语义场景生成方法。其核心目标是生成 真实世界的户外 3D 语义场景，并支持：

  • 场景补全（Semantic Scene Completion, SSC）
  • 场景扩展（Scene Outpainting）
  • 场景修复（Scene Inpainting）

5. 结论

SemCity 采用 Triplane 表示 + 扩散模型 进行 3D 语义场景生成，并支持：

  • 高效 Triplane 结构：减少存储和计算量
  • 扩散模型生成：生成逼真的 3D 语义场景
  • Triplane 操作：支持场景修复、扩展和优化

该方法相比传统 3D 体素生成方法（如 SSD）更加高效，适用于自动驾驶、城市建模等场景。🚀
